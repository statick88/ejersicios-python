# Módulo 9: Testing y QA

## Contenido de esta unidad

### Temas:
- Assert básico
- pytest framework
- Fixtures
- Parametrized tests
- Mocks

### Archivos de esta unidad:
- `tests/m9_testing.py` - Teoría y ejemplos
- `tests/ejercicio_08.py` - Práctica

## Objetivos de aprendizaje

Al finalizar esta unidad podrás:
1. Escribir tests básicos con assert
2. Usar pytest para ejecutar tests
3. Crear fixtures para setup/teardown
4. Usar mocks para simular dependencias

## Cómo ejecutar

```bash
# Instalar pytest
pip install pytest

# Ejecutar tests
python tests/m9_testing.py
python tests/ejercicio_08.py

# O con pytest
pytest tests/ -v
```
