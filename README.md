# Módulo 5: POO - Programación Orientada a Objetos

## Contenido de esta unidad

### Conceptos fundamentales:
- Clases y objetos
- Métodos y atributos
- `__init__` - constructor
- `self` - referencia al objeto

### Pilares de POO:
- **Encapsulación** - proteger datos
- **Herencia** - reusar código
- **Polimorfismo** - múltiples formas
- **Abstracción** - simplificar complejidad

### Contenido acumulado (incluye M0-M4):
- Funciones (M4)
- Control de flujo (M3)
- Operadores (M2)
- Variables y tipos (M1)
- Introducción (M0)

## Objetivos de aprendizaje

Al finalizar esta unidad podrás:
1. Crear clases y objetos
2. Definir métodos y atributos
3. Implementar herencia
4. Usar encapsulación

## Cómo ejecutar

```bash
python POO/persona.py
python POO/herencia/main.py
python POO/encapsulacion/main.py
```
