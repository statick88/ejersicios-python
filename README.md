# Módulo 1: Variables y Tipos de Datos

## Contenido de esta unidad

### Archivos de código:
- `Introduccion/variables.py` - Declarar y usar variables
- `Introduccion/type.py` - La función type() y tipos
- `Introduccion/enteros.py` - Números enteros
- `Introduccion/float.py` - Números decimales
- `Introduccion/boolean.py` - Valores True/False
- `Introduccion/casting.py` - Conversión entre tipos
- `Introduccion/strings.py` - Cadenas de texto

### Contenido aprendido (incluye M0):
- Tu primer programa Python
- Variables y asignación
- Tipos: str, int, float, bool
- Conversión entre tipos (casting)
- Comentarios en código

## Objetivos de aprendizaje

Al finalizar esta unidad podrás:
1. Declarar variables en Python
2. Comprender los tipos de datos básicos
3. Convertir entre tipos (int→str, str→int, etc.)
4. Usar type() para verificar tipos

## Cómo ejecutar

```bash
python Introduccion/variables.py
python Introduccion/casting.py
python Introduccion/strings.py
```
